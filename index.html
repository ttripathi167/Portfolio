Churn and low average ticket

This project focuses on addressing some of the challenges faced by the fictitious company North Winds, including issues such as disorganized data, customer churn, and low average ticket.

To help solve these problems, I employed various Data Science techniques, such as optimizing the database organization, performing churn analysis, conducting cohort and RFM analyses, clustering customers based on their purchasing behavior, implementing a recommendation system, and introducing an Elastic Price strategy.

Through the use of these techniques, we were able to provide valuable insights and solutions for North Winds, enabling them to make data-driven decisions and improve their overall profitability.

Time-Series Regression - Sales Forecast

This project focuses on predicting the sales forecast for the next 6 weeks of the Rossmann pharmacy chain, using Machine Learning models. The models were trained on historical sales data, and the results are delivered through a Telegram chatbot.

Five different models were implemented, and the XGBoost model was selected as the most accurate. To deploy the model into production, Flask and Heroku's cloud were utilized, and the Telegram chatbot was created for users to access the predictions from anywhere.

Overall, this project showcases the potential of Machine Learning in forecasting sales and highlights the benefits of using a chatbot to deliver the results conveniently.

Web Scraping for the Fashion Industry

To gain valuable insights into the fashion market, this project utilized Web Scraping to collect data on products and prices from an e-commerce clothing store. The collected data was then processed using various techniques including Regex, and for the final ETL stage, it was loaded into a SQLite database.

To automate the data collection process, Cron was implemented. Additionally, a dashboard was created using Panel and HVPlot to visualize and explore key insights from the collected data.
